# Defining Program Details

```ruby Runfile
title   'My Utilities'
summary 'Utilities for my Rails'
version '0.1.0'
```

## `title`

=== `title <string>`
Define the title of the application.  
This command is optional.
===


## `summary`

=== `summary <string>`
Define the one line application help message.  
This command is optional.
===


## `version`

=== `version <string>`
Specify a version string for your program.  
This command is optional.
===
